## 🚀 Featured Project

### Job Application Tracker
A full-stack application that helps track job applications with automated email ingestion and status updates.

- **Frontend:** React  
- **Backend:** Node.js + Express  
- **Database:** SQL  
- **Features:** 
  - Background workers with cron jobs
  - Gmail API email parsing
  - Status tracking (Applied, Interview, Rejected)
- **Status:** In development, planning push notifications for real-time updates

🔗 [View Repo](https://github.com/WilliamPottey/job-application-tracker)
